# survivalHelp
Helper functions and workflow to run survival analysis and produce plots and statistics

### Installation:
Needed dependencies:
- survMisc - 0.5.5
- dplyr - 1.0.7
- GSVA - 1.38.2
- survival - 3.2-10

Install survivalHelp package and data:
```
devtools::install_github("hmumme/survivalHelp")
```
